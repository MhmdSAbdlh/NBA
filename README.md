# NBA
NBA Standing (works every year)

Features:
* Ranking 
* Percentage of win 
* Games behind upper 
* Games behind top team 
* Available ranking for each team 
* Final ranking if it is possible 
* Status(Qualify to Playoffs & Play-In tournament)
